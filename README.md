# FACIPA - Facial Paralysis Detection System

A web-based AI application for detecting and analyzing facial paralysis through facial symmetry analysis.

## 🌐 Live Demo
[Visit the live website](https://AKSHITA-tech2.github.io/Facial-Paralysis/)

## ✨ Features
- **Google Authentication** - Secure user login
- **Facial Symmetry Analysis** - AI-powered symmetry scoring
- **House-Brackmann Classification** - Medical-grade assessment
- **Real-time Visualization** - Interactive results display
- **Responsive Design** - Works on all devices

## 🚀 Quick Start
1. Visit the live demo link above
2. Sign in with Google (demo mode)
3. Upload a facial image for analysis
4. View detailed symmetry scores and classification

## 🛠️ Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Python Flask
- **AI/ML**: Computer Vision algorithms
- **Authentication**: Google OAuth
- **Hosting**: GitHub Pages

📊 Analysis Features
Overall facial symmetry scoring

Regional analysis (eyes, brows, mouth)

House-Brackmann grading system

Visual symmetry mapping
